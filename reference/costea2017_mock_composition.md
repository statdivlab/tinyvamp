# The composition of 10 spike-in taxa in 19 samples observed from flow cytometry.

For details, see Costea et al (2017).

## Usage

``` r
costea2017_mock_composition
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 19
rows and 6 columns.
