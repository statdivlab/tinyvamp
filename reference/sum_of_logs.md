# Sum positive numbers via their logarithms Given x and y, this function returns log(exp(x) + exp(y)).

Sum positive numbers via their logarithms Given x and y, this function
returns log(exp(x) + exp(y)).

## Usage

``` r
sum_of_logs(x, y = NULL)
```

## Arguments

- x:

  the thing to be summed.

- y:

  the thing to be added to x. If absent, just the sum over vector-valued
  x is taken.
